# اسم المشروع: Attendance System

## وصف المشروع
مشروع نظام الحضور يهدف إلى متابعة حضور الطلاب/الموظفين وعرض تقارير مفصلة.  
المشروع يستخدم واجهة رسومية مع قاعدة بيانات لتخزين البيانات وإدارة الحضور بشكل فعال.

## طريقة تشغيل المشروع
1. قم بتثبيت XAMPP وتشغيل **Apache** و **MySQL**.  
2. استورد قاعدة البيانات الموجودة داخل مجلد databade.  
3. افتح المشروع باستخدام Visual Studio (لـ C#) أو محرر الأكواد المناسب.  
4. تأكد من تعديل إعدادات الاتصال بقاعدة البيانات إذا لزم الأمر.  
5. شغّل المشروع وجرب الوظائف (إضافة حضور، عرض تقارير، إلخ).  

## ملاحظات
قبل تشغيل المشروع في فيجوال استوديو يجب عمل Build Solution حتى يتم انشاء ملفات التشغيل لانه تم رفع المشروع الى جيت هب بدون ملفات التشغيل
